## Sobre

A **Uranus** não vende esforço. Constrói **sistemas**.

Somos uma empresa de infraestrutura tecnológica sediada em **Curitiba, Paraná**, operando em **dois continentes** e entregando para **seis verticais diferentes**. Nosso trabalho é transformar operações empresariais em plataformas previsíveis, escaláveis e auditáveis — usando IA aplicada, automação inteligente, engenharia de software e transformação cloud como as ferramentas.

> **Antes de ser grande, uma empresa precisa ser precisa.**
> **Antes de escalar, precisa ser previsível.**
> **Nós construímos a base para os dois.**
